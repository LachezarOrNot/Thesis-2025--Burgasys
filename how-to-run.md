# Как да се пусне

## 1) Подготовка

1. Инсталирайте [Node.js](https://nodejs.org/) (препоръчително LTS).
2. Отворете терминал в папката на проекта.
3. Инсталирайте зависимостите:

```bash
npm install
```

## 2) Настрой .env

Проверете дали в `.env` има нужните `VITE_*` променливи (Firebase, Agora, Gemini).

## 3) Пуснете проекта

```bash
npm run dev
```

След това отворете адреса, който Vite покаже в терминала (`http://localhost:5173`).

Хостнатият проект е на адрес https://burgasys.web.app
